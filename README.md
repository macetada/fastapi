# `macetada/fastapi`

Repositório para macetar o FastAPI com requests.

## Subindo o server

```bash
docker build -t fastapi .
docker run --rm -it -p 8081:80 fastapi
```

> Este procedimento deve ser feito numa máquina servidora ou num cluster.
